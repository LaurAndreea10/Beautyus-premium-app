## 🧩 Core Modules

### 1. Client Experience
The project presents a **premium service catalog** with clear service structure, duration, and indicative pricing for categories such as:

- Hair Styling
- Color & Balayage
- Nails
- Make-up
- Facial Treatments
- Brows & Lashes :contentReference[oaicite:4]{index=4}

### 2. Booking & Calendar
The booking section includes:

- service selection
- stylist selection
- date input
- client details
- available time slots
- booking notes
- confirmation flow :contentReference[oaicite:5]{index=5}

It also includes a **calendar overview** with an hourly agenda for team members and rooms, showing statuses such as **Booked**, **Available**, and **Pending**. :contentReference[oaicite:6]{index=6}

### 3. Operations Console
The internal CRM area includes a **Client Desk** with inline editing for:

- name
- email
- phone

It also includes export actions such as **JSON** and **CSV**, plus client-level data like service, stylist, status, spend, and next appointment. :contentReference[oaicite:7]{index=7}

### 4. Automation Layer
The live app showcases retention-focused automation flows, including:

- automatic reminder 24h before appointment
- post-visit message for review and feedback
- return reminder after 4–6 weeks
- internal alert for inactive VIP clients :contentReference[oaicite:8]{index=8}

### 5. Revenue Pipeline
The pipeline is structured as:

**Lead → Booking → Follow-up → VIP retention** :contentReference[oaicite:9]{index=9}

### 6. Client Profile
A complete client profile view includes:

- contact details
- booking status
- last visit
- current service
- lifetime spend
- retention labels
- client notes
- service history
- actions like booking, reminder, note, and invoice :contentReference[oaicite:10]{index=10}
